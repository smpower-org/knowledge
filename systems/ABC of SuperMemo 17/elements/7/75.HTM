Try to import an article now!

For example:

  1. Open any article in your web browser (for example, right click over this link www.super-memory.com/help/read.htm and choose Open in new window)
  2. Select any portion of text with the mouse (in the browser)
  3. Press Ctrl+C (in the browser)
  4. Return to SuperMemo (e.g. with Alt+Tab)
  5. Press Ctrl+N (in SuperMemo)

Hint #1: After you have imported an article, you can always return to this page by using the button (Alt+Left arrow).

Hint #2: You can also import articles directly from Wikipedia. If Internet Explorer is your default browser*:
  1. press Ctrl+Shift+W,
  2. type SuperMemo, and
  3. press Enter to import an article about SuperMemo from Wikipedia.

* If you use another browser, you will need to first open a Wikipedia article in Internet Explorer, and only then press Ctrl+Shift+W.
